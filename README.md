## Showcase Game — Tic-Tac-Toe (Unbeatable AI)
Path: `games/tictactoe/`

Yeh ek simple web-based Tic-Tac-Toe game hai jisme aap computer ke khilaaf khel sakte hain. AI minimax algorithm use karta hai, isliye AI ko harana mushkil hai — accha showoff piece.

Run karne ka tareeka:
- Local file ke through: `games/tictactoe/index.html` ko browser mein open karein.
- Ya ek simple web server chalayein:
  - Python 3: `python -m http.server` aur phir browser mein `http://localhost:8000/games/tictactoe/`
  - Ya koi bhi static server use kar sakte hain.
